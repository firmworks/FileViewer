Post Install Config

  1. Add the 'Tag and Upload' Global Action to the Global layout or to the layout of any Object

  2. Add the Lightining Component "File Viewer" to any lightning Record Page Layout to get a contextual view of related content.

  3. Curate taxonomy fields by adding custom fields onto the ContentVersion Object to control tagging and searchablity. Add picklists, multipicklists, lookup fields, and others to help define the shape of the documents.

  4. Give users permissions to the taxonomy fields and the permission set for 'FileViewer'.

  5. Discover and work with files through the File Viewer tab.
